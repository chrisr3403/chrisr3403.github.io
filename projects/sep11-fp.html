Game Editor DX (FP Project)

What is Game editor dx?

The Game Editor DX is a game system made from Kaboom to make it so you can switch characters in the game. Games such as sonic where if you want a different character you press a key to switch from Knuckles, Tails or supersonic..

Making the game editor will give new content and scene for people that play it. It can give gamers a type of content so they cant just play one character but multiple while in game! The editor is designed for also platforming so many people can play the platformer esc game. This game editor is based off sonic and I used kaboom to make this editor possible..
